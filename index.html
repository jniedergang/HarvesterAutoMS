Windows Autounattend Generator

Basic Parameters

▼

Installation Source

▼

Choose where the Windows installation ISO comes from. This is informational — it does not affect the autounattend XML but is saved in the config for reference during Terraform provisioning.

Place Windows ISOs in the images/ directory. They will be served over HTTP.

Drivers

▼

Select which driver sources to include in the autounattend ISO. Drivers are copied into $WinPEDriver$ and auto-loaded by Windows PE.
